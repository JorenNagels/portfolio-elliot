# Hero image

Drop **one** file in this directory:

- `hero.jpg` — large background portrait used behind the hero section.
  - Recommended size: ~1920×1280 (landscape) or ~1600×2000 (portrait crop), JPG, < 400 KB after compression.
  - The image is darkened with a left-to-right gradient overlay, so the subject should sit on the right half of the frame for best legibility.

The site references this file as `/images/hero/hero.jpg` (see `src/components/Hero.tsx`).
